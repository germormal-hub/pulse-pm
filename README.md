# Pulse PM

Modern project management tool with extensive dashboards and reporting. Built with React + Vite.

## Features

- **Dashboard** — KPI cards, status distribution donut chart, team velocity, sprint burndown, project health, activity feed
- **Kanban Board** — Drag-style columns with priority indicators, progress tracking, and team avatars
- **Timeline** — Gantt-style view with progress overlays
- **Reports** — Completion rate, cycle time, efficiency trends, team performance rankings, priority breakdown
- **Team** — Individual member cards with workload distribution and active tasks

## Tech Stack

- React 19
- Vite 6
- Custom SVG charts (no dependencies)
- DM Sans typography

## Getting Started

```bash
npm install
npm run dev
```

## Deploy to Vercel

1. Push this repo to GitHub
2. Go to [vercel.com](https://vercel.com) and import the repo
3. Vercel auto-detects Vite — click Deploy
4. Done. Your app is live.
